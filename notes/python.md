# Jupyter Notebook
## Shortcuts
- **Run cell**: `Shift + Enter`
- **Insert cell below**: `B`
- **Insert cell above**: `A`
- **Delete cell**: `D, D`
- **Undo cell deletion**: `Z`
- **Move cell up**: `K`
- **Move cell down**: `J`
- **Copy cell**: `C`
- **Paste cell**: `V`
- **Cut cell**: `X`
- **Select cell**: `Shift + Space`
- **Deselect cell**: `Shift + Space`
- **Edit cell**: `Enter`
- **Save notebook**: `Ctrl + S`
- **Stop cell**: `Ctrl + M, I`
- **Restart kernel**: `0, 0`
- **Interrupt kernel**: `I, I`
- **Change to code cell**: `Y`
- **Change to markdown cell**: `M`
- **Change to raw cell**: `R`

### Types of Data
1. Number and Boolean Values
- Integers = Positive or negative whole numbers
- Floats = Decimal numbers
- Booleans = True or False
```
x1 = 5
type(x1)
```
> int
```

```
> 2.0
```

```

2. Strings
- sequence of characters
```

```
